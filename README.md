# 💼 Salary Prediction App

## 📌 Overview

This is an end-to-end Machine Learning project that predicts salary based on years of experience using Linear Regression. The model is deployed as a Streamlit web application for easy interaction.

---

## 🚀 Features

* Predict salary based on experience
* Simple and interactive web interface
* Real-time predictions

---

## 🛠️ Tech Stack

* Pandas
* NumPy
* Scikit-learn
* Streamlit
* Joblib

## ▶️ How to Run Locally

1. Clone the repository:

```bash
[git clone https://github.com/YOUR-USERNAME/salary-prediction-streamlit-app.git](https://github.com/mrupdateabdullahipython/salary-prediction-streamlit-app
```

2. Navigate to the project folder:

```bash
cd salary-prediction-streamlit-app
```

3. Install dependencies:

```bash
pip install -r requirements.txt
```

4. Run the app:

```bash
streamlit run app.py
```
